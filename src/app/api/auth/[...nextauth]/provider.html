üèóÔ∏è Credentials (Uppercase)
Credentials is a function imported from Auth.js that creates a custom authentication provider.
import Credentials from "next-auth/providers/credentials" // It's a factory function that returns a provider const myProvider = Credentials({ // configuration goes here })
Think of it as a "provider builder" - you give it configuration and it builds a working authentication provider for you.
üìù credentials (lowercase)
The credentials property defines what form fields the user will see and fill out.
credentials: { email: { label: "Email", type: "email" }, password: { label: "Password", type: "password" } }
This creates a login form with email and password fields. The keys (email, password) become the property names you'll receive in the authorize function.
üîç authorize Function
The authorize function is your custom logic that verifies if the user's credentials are valid.

It receives the form data and must return either a user object (success) or null/throw error (failure).
